## 209.长度最小的子数组
给定一个含有 n 个正整数的数组和一个正整数 s ，找出该数组中满足其和 ≥ s 的长度最小的连续子数组，并返回其长度。如果不存在符合条件的子数组，返回 0。
示例：
输入：s = 7, nums = [2,3,1,2,4,3]
输出：2
解释：子数组 [4,3] 是该条件下的长度最小的子数组。
提示：
1 <= target <= 10^9
1 <= nums.length <= 10^5
1 <= nums[i] <= 10^5

这道题使用了滑动窗口的思想，滑动窗口和双指针有点像，时间复杂度为O(n)我在做题时发现了以下要点：
1. 可以使用```return result == Integer.MAX_VALUE ? 0 : result;```这样的代码来判断是否进入循环，从而确定返回结果。
2. 滑动窗口的要点是确认右边界，滑动左边界。若是确认左边界，滑动右边界就和暴力做法没区别了。

59.螺旋矩阵II
